# QR Code Generator

A simple and responsive **QR Code Generator** web application built with **HTML, CSS, JavaScript, and Bootstrap**. This project allows users to generate QR codes from any URL or text and download them as PNG files with a custom file name.

## Features

- Generate QR codes from any URL or text input.  
- Custom file name for downloaded QR code.  
- Fully responsive and works on desktop, tablet, and mobile.  
- Clean and modern UI design with shadows, rounded corners, and hover effects. 



## Technologies Used

- HTML5  
- CSS3 (with modern styling and responsive design)  
- JavaScript  
- [Bootstrap 5](https://getbootstrap.com/)  
- [QRCode.js](https://github.com/davidshimjs/qrcodejs)
